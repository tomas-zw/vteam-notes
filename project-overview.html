1. Översikt

1.1. Webbaserad lösning för backend med koppling mot databas för att administrera

  • cyklar
  • laddstationer
  • parkeringszoner
  • tillåtna zoner att cykla i
  • utlåning/återlämning
  • hämtning av cykel

1.2. REST API med möjlighet att koppla in anpassade applikationer.

  • Väldokumenterat REST API
  • Hantera flera olika versioner, tex genom att använda v1/ som en del i URI:n.
  • Hantera autentisering

  • Hantera relevant data.
